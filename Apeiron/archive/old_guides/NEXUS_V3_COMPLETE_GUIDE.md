## 🚨 Troubleshooting

### **"ModuleNotFoundError: No module named 'chromadb'"**
```bash
pip install chromadb
```

### **"Rate limit exceeded" (ArXiv)**
```python
# In nexus_ultimate_v3.py, increase sleep:
time.sleep(5)  # Was: 2
```

### **"Memory error" (ChromaDB)**
```bash
rm -rf nexus_memory/.chroma.lock
```

### **Dashboard not updating**
```bash
# Check if state file exists:
ls -la nexus_ultimate_state.json

# Check if main engine is running:
ps aux | grep nexus_ultimate
```
