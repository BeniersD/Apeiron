## How Can I Contribute?

APEIRON is an open research programme. You can contribute in several ways:

- **Code**: implement new layers, improve existing algorithms, fix bugs.
- **Documentation**: improve docstrings, write tutorials, fix typos.
- **Testing**: write unit tests, integration tests, performance benchmarks.
- **Research**: propose new atomicity frameworks, validate hypotheses, publish companion papers.
- **Review**: provide feedback on pull requests and architectural discussions.

If you are new to the project, look for issues labeled `good first issue` or `help wanted`.
