\begin{abstract}
We present a multi-prover \emph{architecture} for formally verifying the
atomicity—the irreducibility—of elementary representational units
within the \textsc{Apeiron} Framework. The architecture integrates four
theorem provers (SymPy, Z3, Lean~4, and Coq) into a unified pipeline,
\texttt{self\_proving.py}, that generates proof obligations, executes
them, and exports JSON certificates.

A rigorous evaluation on a benchmark of 47 observables reveals the
current formal coverage: \textbf{Z3} correctly decides Boolean atomicity
for integers and sets; \textbf{Lean~4} and \textbf{Coq} provide genuine
proofs for integers via computational reflection, but for all other
observable types and frameworks their backends are still placeholders;
\textbf{SymPy} serves as a fast consistency filter but is not sound.
We formalise this situation as a \emph{trust hierarchy} and generate
certificates that faithfully reflect it.

The contribution of this paper is the architecture itself, together
with an honest characterisation of what each prover can currently
deliver. We discuss the semantic gap induced by string‑templating,
propose an AST‑based translation layer as the next step, and outline
how the Lean/Coq backends can be extended to broader classes of
observables.
\end{abstract}
